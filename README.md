# TaskMaster Pro

**TaskMaster Pro** is a modern task and workflow management system built using the Onion Architecture in ASP.NET Core and Angular.

It showcases scalable enterprise architecture, clean code, layered separation of concerns, JWT authentication, and a clean Angular frontend.

## 🛠 Tech Stack

- ASP.NET Core 7 / 8 (Web API)
- Angular 16+
- Entity Framework Core
- MSSQL Server
- Onion Architecture
- SignalR (for real-time notifications)
- JWT Authentication & Role-based access control

## 🚀 Key Features

- User Registration & Role-based Login
- Create & Manage Projects and Tasks
- Assign tasks to users
- Dashboard with task statistics
- Real-time task updates (SignalR)
- Export tasks to Excel or PDF
- Responsive Angular UI
